Descarga la última versión de MiApp y sigue las instrucciones para instalar y usar la aplicación correctamente.

Descargar VRPOptimax
1. Descargar archivo de prueba
Primero, descarga el archivo Excel de prueba. Este archivo contiene la estructura de datos para ingresar los puntos (coordenadas), vehículos y nombres de los puntos.
2. Llenar los datos
Abre el archivo Excel y llena los datos con los puntos, vehículos y nombres correspondientes.
3. Cargar archivo en VRPOptimax
Una vez que tengas los datos, abre MiApp y carga el archivo Excel en la aplicación.
4. Optimizar rutas
Haz clic en el botón "Optimizar rutas" para que la aplicación procese la información. Ten en cuenta que a mayor cantidad de rutas, mayor será el tiempo de procesamiento. Se recomienda no superar los 200 puntos para mantener un tiempo razonable.
5. Visualizar resultados
Después de procesar los datos, podrás ver las rutas optimizadas en la tabla inferior. El botón "Ver en mapa" mostrará la ruta seleccionada en Google Maps. Debajo también encontrarás un mapa con todas las rutas creadas y un filtro para seleccionar cada una.
